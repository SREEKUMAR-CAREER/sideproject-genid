# ID Card SaaS

AI-Powered Employee ID Card Automation SaaS.

## Folder Structure

- `frontend/`: Next.js 14 App Router application.
- `firebase/`: Firebase Cloud Functions and Rules.
- `docs/`: Deployment and API documentation.

## Setup

1. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Firebase**:
   ```bash
   cd firebase/functions
   npm install
   npm run build
   ```

## Development

See `frontend/README.md` for more details on the UI.
See `firebase/functions/README.md` for backend logic.
